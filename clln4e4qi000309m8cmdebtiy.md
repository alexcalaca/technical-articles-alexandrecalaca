## Check Konsole installation

### Check version

```apache
konsole --version
```

Your output might look like:

```apache
konsole 18.04.0
```

---

### Check

The `which` command will display the path to the executable file associated with a given command.

IN your terminal:

```apache
which konsole
```

Your output might look like:

```apache
calaca@calaca-PC ~ $ which konsole
/usr/bin/konsole
```
